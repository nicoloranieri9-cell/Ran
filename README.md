# Registro Veicoli

Piccola app web per gestire le scadenze del proprio veicolo.

## Funzionalità
- Piano Base (gratis): 1 veicolo, revisione e assicurazione.
- Piano Premium (5,90€/anno): più veicoli, bollo e tagliandi.
- Salvataggio locale in `localStorage`.
- Evidenza stato scadenze (ok / in scadenza / scaduta).

## Avvio rapido
Apri `index.html` nel browser.
